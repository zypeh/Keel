## Introduction
**Keel** is a programming language, aiming to make programmers more productive and feel smart.

## Building & Running
To build this project you need Rust and Cargo.

```sh
git clone https://github.com/zypeh/keel.git
cd keel

# Install or build
cargo build
cargo install
```
